# PR evidence templates

The canonical reusable templates now live in [Reference — PR evidence templates](reference/pr-evidence-templates.md).

Use that page for:

- the full evidence-pack structure;
- documentation-only changes;
- workflow changes;
- publishing or deployment changes;
- remediation evidence; and
- compact-versus-full selection rules.

## What remains true

Every template preserves the same core evidence:

- linked execution contract;
- changed and deliberately excluded work;
- acceptance-criteria treatment;
- validation completed, pending or not performed;
- post-merge verification;
- risks and caveats;
- groundedness review; and
- one final recommendation.

Do not pre-check evidence that has not run. A template shortens presentation; it does not reduce the validation, review or human-authority requirements.
